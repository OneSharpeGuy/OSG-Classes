FlashID

[FlashID]

Synopsis

Generates a unique Instance ID

Description

Generates a unique instance identifier based on elapsed time since a specified epoch, ensuring uniqueness across different systems.

The structure of FlashID is

4EBF-B06A-LQNO

HexDate 4-character hexadecimal representation of the number of days elapsed since the specified epoch. (The default is 1/1/1970, the UNIX epoch).
Partial Day A time-based component representing the elapsed time in the current day.
Adjunct A 4-character random alpha string ensuring uniqueness across different installations.
The Adjunct string does not contain digits or characters used in hexadecimal notation (`0-9`, `A-F`).
